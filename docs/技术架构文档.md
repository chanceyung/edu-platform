# 智慧教学平台 · 技术架构文档

## 系统概览

面向初中学校的 AI 教学减负系统。10 后端模块 + 10 前端页面 + 4 个真实智谱 GLM-5.2 AI 能力。

## 技术栈

| 层 | 技术 |
|---|---|
| 后端 | Spring Boot 3.2 + JDK 17 + MyBatis-Plus 3.5 + MySQL 8 + Redis |
| 前端 | Vue 3 + TypeScript + Element Plus + Vite + Pinia + Vue Router |
| AI | 智谱 GLM-5.2（批改/评语/备课/组卷）+ JDK HttpClient + ai_task_log 信任内核 |
| 部署 | Docker Compose（MySQL/Redis/MinIO/后端/前端） |
| 监控 | Prometheus |

## 后端模块（10个）

| 模块 | 职责 | 依赖 |
|---|---|---|
| common | BaseEntity/R/MetaObjectHandler | — |
| gateway | 启动入口 | 所有模块 |
| school-org | 学校/年级/班级/学生 | common |
| question-bank | 题库/选项 | common |
| exam-engine | 试卷/组卷/考试/答卷/判分 | common, question-bank, school-org |
| homework | 作业布置/提交/批改 | common, school-org |
| ai-service | 智谱GLM网关/AI批改/AI评语/AI备课/AI组卷/信任内核 | common, homework, exam-engine |
| analytics | 班级分析/个人学情/错题本/校领导看板 | common, exam-engine, homework, question-bank |
| home-school | 家校通知 | common |

## AI 能力（4个，全部真实智谱 GLM-5.2 验证）

| 能力 | 模块 | API | 信任内核 |
|---|---|---|---|
| AI 批改 | AiGradingService | POST /ai/grading/grade | ai_task_log(GRADING) |
| AI 评语 | AiCommentService | POST /ai/comment/generate | ai_task_log(COMMENT) |
| AI 备课 | AiLessonService | POST /ai/lesson/generate | ai_task_log(LESSON) |
| AI 智能组卷 | AiComposeService | POST /ai/compose/auto | — |

## 数据库（13张表）

org_school / org_grade / org_class / org_student
qbank_question / qbank_option
exam_paper / exam_paper_question / exam / exam_record / exam_record_answer
homework / homework_submission
ai_task_log / hs_notice

## 设计规范（CLAUDE.md 铁律）

- 千分制分值（score×1000）
- tenant_id 自动填充（MetaObjectHandler strictInsertFill）
- 审计字段（createTime/updateTime/creator/updater/isDeleted）
- AI 必可追溯（ai_task_log）
- TDD（16+ 单元测试）
- License 纯净（学之思只参考设计不抄码）

## 运行

```bash
# 后端
cd backend && mvn clean install && java -jar gateway/target/gateway.jar
# 前端
cd frontend/teacher-workbench && pnpm install && pnpm dev
# 基础设施
cd deploy && docker compose up -d mysql redis minio
# 验证
curl http://localhost:18080/api/v1/org/schools
# 浏览器
http://localhost:5173
```
